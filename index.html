jQuery.miniModal plugin

A simple but dynamic modal popups plugin. ("simple" being a relative term of course)

This plugin is a simple to use but very specific implementation classical type of modal popup windows.

It dynamically creates modal popups with a title, a body and and ok and/or cancel. miniModal also provides the necessary events to create those popups in a repeatable manner. So, if you have 12 instances of a "about him" popup on a page. The dynamic hooking to all those buttons and how each modal can open and close is already available as easy to change parameters.

Sample

This sample provides "edit" button to a list of poem as a degradable javascript enhancement. The behavior is added dynamically and each modal windows hook themselves automatically.

Plugin options and setup

Note that this plugin requires jqModal and can be optionnaly be coupled to jquery.Metadata.

More configuration options are available, refer to the plugin's default settings for reference.

Javascript code to setup and initialize the sample

$(document).ready(function(){
	$(".poem").each(function(){
		$this = $(this);
		poemId = $("pre", $this)[0].id;
		$this.append($('<button id="btnEdit-' + poemId + '" class="editPoemButton {poemId:' + "'" + poemId + "'" + '}">Edit</button>'))
	});

	$(".editPoemButton").miniModal({
		overlay: 10,
		onBeforeShow: function(hash, $this) {
			var poem = $("#" + $this.metadata.poemId).html();
			$(".editPoem", $this).val(poem);
			return true;
		},
		onOk: function($this) {
			$("#" + $this.metadata.poemId).html( $(".editPoem", $this).val() )
			return true;
		},
		templateWindow: function($this) {
			// Returns a custom template taken from a hidden div in the dom
			var newModal = $("#editPoemModalTemplate").clone();
			newModal[0].id = "";
			return newModal;
		}
	});
})
